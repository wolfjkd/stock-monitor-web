# Stock Monitor Web v3.0 - A股价格预警监控系统（Web版）

Flask + Bootstrap 图形界面，基于CLI版核心代码重构

## 版本定位

| 项目 | 版本 | 说明 |
|------|------|------|
| [stock-monitor](https://github.com/wolfjkd/stock-monitor) | CLI版 v2.0 | 命令行工具，轻量高效 |
| **stock-monitor-web** | Web版 v3.0 | 图形界面，已开发完成 |

## 项目状态

✅ **已开发完成** - 2026-06-08

## 核心功能

### 实时行情模块
- 展示所有监控股票的实时价格
- 显示涨跌幅、成交量等关键指标
- 颜色标识：红色上涨、绿色下跌（中国习惯）
- 状态标识：安全/临界/跌破
- 每5秒自动刷新

### 监控配置模块
- 查看当前监控列表
- 在线修改目标价
- 添加新股票到监控列表
- 从监控列表删除股票
- 修改监控方向（跌破/涨破/双向）
- 配置修改即时生效
- 委托方向显示（多仓/空仓）

### 监控控制模块
- 一键启动监控
- 一键停止监控
- 显示监控运行状态
- 显示监控运行时长
- 显示数据源信息

### 预警日志模块
- 实时显示预警触发记录
- 滚动显示最新日志
- 日志持久化存储
- 支持日志清空

## 技术栈

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 后端框架 | Flask | 轻量级，快速开发 |
| 前端框架 | Bootstrap 5 | 现成组件，美观 |
| 实时通信 | SSE | Server-Sent Events，简单实现推送 |
| 数据源 | eltdx + 腾讯接口 | 复用CLI版核心代码 |
| 进程管理 | Python threading | 复用CLI版监控逻辑 |

## 修改历史

### 2026-06-08 (v3.0 正式版)

#### 后端开发
- Flask后端API：实时行情、监控配置、预警日志、SSE推送
- 多TDX节点支持：故障转移、手动切换
- 自动启动监控，5秒刷新行情

#### 前端开发
- Bootstrap 5前端页面，适配1920x1080单屏显示
- 实时行情卡片横排显示
- 监控配置表格：9列（股票、代码、涨跌幅、当前价、目标价、委托、状态、监控方向、操作）
- 不同股票用不同背景色+分隔线区分
- 列与列之间加分隔线
- 预警触发时表格行和行情卡片闪烁

#### 预警逻辑
- 临界预警阈值：0.01元（绝对值）
- 委托方向：多仓（红色加粗）、空仓（绿色加粗）
- 预警日志实时滚动显示

#### 启动方式
- `run.bat` 双击启动（Windows）
- 命令行：`python app.py`
- 浏览器访问：http://localhost:5000

## 代码复用

本项目基于CLI版核心代码重构，复用以下模块：

| 现有代码 | 复用方式 | 改造量 |
|----------|----------|--------|
| get_quote_tdx() | 直接调用，获取实时行情 | 无 |
| get_quote_tencent() | 备用数据源 | 无 |
| watchlist_config.json | 读写配置文件 | 无 |
| check_single() | 重构为Web版监控逻辑 | 中等 |
| check_multi() | 重构为Web版监控逻辑 | 中等 |
| show_popup() | 改为Web通知/日志 | 简单 |
